# Quartier

> **A home for your research.**
>
> *This line was added to test the* ***commit*** *functionality of the application.*

Quartier is a collaborative, web-based editor for Quarto documents. It combines the power of markdown editing with real-time collaboration, making it easy to work on research papers, reports, and documentation.

## Features

* **Dual-Mode Editing** — Switch between a visual WYSIWYG editor and source code view

* **Real-Time Collaboration** — Work together using WebRTC-powered sync (Yjs)

* **Command Palette** — Quick access to actions with `Ctrl+K` / `Cmd+K`

* **File Explorer** — Browse and manage your Quarto project files

* **GitHub Integration** — Commit changes directly to your repository

## Tech Stack

* **Vue 3** + TypeScript + Vite

* **Milkdown** — WYSIWYG Markdown editor

* **CodeMirror** — Code/source editing

* **Yjs + y-webrtc** — Collaborative editing

* **shadcn-vue** — UI component library

* **Tailwind CSS** — Styling

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```
